Dataset Title:
Anonymized Clinical Dataset for Prediction of In-Hospital Cardiac Arrest in Traumatic Spinal Cord Injury Patients

Description:
This dataset contains de-identified clinical data collected from ICU patients with traumatic spinal cord injury. All direct identifiers were removed in accordance with institutional ethics approval.

Variables include neurological injury level, paralysis type, injury mechanism, vasopressor use, APACHE II score, and laboratory/physiological measurements.

Outcome variable represents in-hospital cardiac arrest (IHCA).

Time variables and identifiers were excluded to ensure anonymity.

See "data_dictionary.xlsx" for detailed variable definitions.
